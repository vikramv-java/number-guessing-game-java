# Number Guessing Game - Java

A console-based Number Guessing Game built using Java.

## Features

* Multiple difficulty levels
* Random number generation
* Score system
* Best score saving using file handling
* Input validation
* Replay option
* Exception handling

## Technologies Used

* Java
* File Handling
* OOP Concepts
* Scanner Class
* Random Class

## Difficulty Levels

* Easy → 1-50 numbers, 10 attempts
* Medium → 1-100 numbers, 7 attempts
* Hard → 1-100 numbers, 5 attempts

## How to Run

1. Compile the program:
   javac NumberGuessingGame.java

2. Run the program:
   java NumberGuessingGame

## Project Purpose

This project was created to practice Java fundamentals, logic building, file handling, and console application development.
